# SAFE Grant Writing System - New RFP Processing Workflow

## Quick Start Guide

**When you have a new grant opportunity, provide Claude with either:**
1. **File path to RFP document**: `/Users/louispiotti/Desktop/New_RFP.pdf`
2. **URL to grant webpage**: `https://foundation.org/grants/rfp-2025`

Claude will automatically analyze the opportunity and create a complete application package using this system.

## System Overview

This workflow leverages your comprehensive grant writing system including:
- **Knowledge Base**: Complete SAFE organizational profile and program models
- **Research Library**: Current statistics and evidence-based research
- **Template System**: Proven grant application structures
- **MCP Servers**: For research, document creation, and file management

## Step-by-Step Workflow

### Phase 1: RFP Analysis & Setup (10-15 minutes)

#### 1.1 RFP Document Analysis
Claude will:
- Extract all requirements, deadlines, and submission guidelines
- Identify required attachments and supporting documents
- Analyze evaluation criteria and scoring rubrics
- Note any unique requirements or restrictions

#### 1.2 Project Folder Creation
Automatic creation of organized folder structure:
```
/Users/louispiotti/llm-training/SAFE-Grant-Writing-System/Active-Grants/
├── [FUNDER_NAME_YEAR]/
    ├── 01_RFP_Documents/
    ├── 02_Research_Notes/
    ├── 03_Draft_Documents/
    ├── 04_Final_Submission/
    └── 05_Supporting_Materials/
```

#### 1.3 Requirements Checklist
Claude creates a comprehensive checklist including:
- Application narrative sections
- Required attachments (budgets, letters, etc.)
- Formatting requirements
- Submission deadlines
- Contact information

### Phase 2: Research & Content Development (30-45 minutes)

#### 2.1 Targeted Research
Using your MCP servers, Claude will:
- Search for funder-specific information and priorities
- Gather relevant statistics and research studies
- Identify successful similar programs
- Research evaluation methods and best practices
- Save all findings to the project research folder

#### 2.2 Program Matching
Claude analyzes the RFP against SAFE's knowledge base to:
- Identify which SAFE programs align with funder priorities
- Determine appropriate service areas and target populations
- Select relevant partnerships and collaborations
- Choose optimal budget structure based on similar past grants

#### 2.3 Narrative Outline Creation
Development of detailed outline including:
- Need statement with local/relevant statistics
- Program description using SAFE's proven models
- Outcomes and evaluation plan
- Sustainability strategy
- Logic model or theory of change

### Phase 3: Document Creation (45-60 minutes)

#### 3.1 Core Narrative Writing
Claude creates grant narrative using:
- SAFE's established voice and writing style
- Funder-specific language and priorities
- Current research and statistics
- Proven program models from knowledge base

#### 3.2 Supporting Document Creation
Automatic generation of:
- **Detailed Budget**: Line-item budget with narrative justification
- **Logic Model**: Visual representation of program theory
- **Evaluation Plan**: Comprehensive assessment strategy
- **Sustainability Plan**: Long-term funding and continuation strategy
- **Timeline**: Project implementation schedule

#### 3.3 Required Attachments
Claude prepares templates for:
- Letters of support (with draft language)
- Organizational chart and staff qualifications
- 990 tax returns and financial statements (guidance on gathering)
- Board of directors list
- Any additional funder-specific requirements

### Phase 4: Quality Review & Refinement (20-30 minutes)

#### 4.1 Internal Consistency Check
Claude reviews all documents to ensure:
- Budget aligns with narrative activities
- Timeline is realistic and achievable
- Evaluation plan matches stated outcomes
- All RFP requirements are addressed

#### 4.2 SAFE Standards Compliance
Verification that application includes:
- Accurate organizational information
- Consistent program descriptions
- Appropriate partnership details
- SAFE's key differentiators and strengths

#### 4.3 Final Polish
Claude performs final review for:
- Grammar, spelling, and formatting
- Professional tone and clarity
- Funder-specific customization
- Submission guideline compliance

### Phase 5: Submission Preparation (15-20 minutes)

#### 5.1 Final Document Package
Claude creates organized submission package with:
- Cover letter (if required)
- Complete application narrative
- All required attachments
- Submission checklist verification

#### 5.2 Follow-up Action Items
Claude provides:
- List of items requiring your personal attention
- Contacts for letters of support
- Submission deadline reminders
- Post-submission follow-up schedule

## Common Grant Types and Approaches

### Federal Grants (SAMHSA, CDC, etc.)
- Emphasis on evidence-based practices
- Detailed evaluation and data collection
- Cultural competency requirements
- Sustainability planning

### Foundation Grants
- Mission alignment crucial
- Relationship building important
- Impact measurement focus
- Innovation and creativity valued

### Corporate Grants
- Business case presentation
- Community benefit emphasis
- Partnership opportunities
- ROI demonstration

### State/Local Grants
- Local needs assessment critical
- Community partnerships required
- Geographic service area specific
- Local data and statistics important

## Tips for Success

### Research Phase
- Always use the most current statistics available
- Look for funder's previously funded projects
- Identify potential collaborators early
- Understand funder's language and priorities

### Writing Phase
- Use SAFE's authentic voice consistently
- Include specific, measurable outcomes
- Provide clear logic connecting activities to outcomes
- Address sustainability from the beginning

### Review Phase
- Read from funder's perspective
- Verify all requirements met
- Check for internal consistency
- Proofread carefully

## Emergency/Rush Applications

For applications needed quickly (less than 1 week):
1. **Immediate Analysis**: Prioritize RFP analysis and requirements extraction
2. **Expedited Research**: Focus on essential statistics and evidence
3. **Template Utilization**: Leverage existing successful applications
4. **Streamlined Review**: Focus on requirements compliance
5. **Quick Turnaround**: Prioritize submission over perfection
