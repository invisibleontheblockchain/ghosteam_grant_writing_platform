# SAFE Grant Application Template & Workflow

## Pre-Application Checklist

### Initial Assessment
- [ ] Grant opportunity aligns with SAFE's mission
- [ ] Eligible applicant (501(c)(3), location requirements, etc.)
- [ ] Funding amount appropriate for program scale
- [ ] Timeline feasible for implementation
- [ ] Required match or cost-share achievable
- [ ] Partners available and committed

### Required Documents Inventory
- [ ] Current board list with required information
- [ ] Most recent 990 tax filing
- [ ] Financial audit/review/statements (past 2 years)
- [ ] Current year operating budget
- [ ] IRS determination letter
- [ ] Certificate of insurance
- [ ] Organizational chart
- [ ] Annual report (if applicable)

## Writing Tips & Reminders

### DO:
- Use active voice
- Include specific numbers and targets
- Connect to funder priorities
- Demonstrate cultural competence
- Show evidence base
- Emphasize lived experience
- Highlight innovation
- Include student/community voice

### DON'T:
- Use jargon without explanation
- Make unsupported claims
- Forget about sustainability
- Ignore page/word limits
- Submit without proofreading
- Miss deadlines
- Forget required signatures
